# @nexwave/cli

Command-line interface for the Nexwave execution kernel.

> ⚠️ **Coming Soon** — This package is a placeholder. CLI implementation is in progress.

## Planned Features

```bash
nexwave plan agent.yaml      # Show execution plan
nexwave apply agent.yaml     # Execute intent/agent
nexwave status               # Global status
nexwave agent list           # List agents
nexwave exec kill            # Emergency stop
```
